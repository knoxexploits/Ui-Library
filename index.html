Open SRC UI ||Quick Start

Load the library using the official loader URL.

local RedGlass = loadstring(game:HttpGet("https://raw.githubusercontent.com/knoxexploits/Ui-Library/refs/heads/main/RedGalss"))()

local Window = RedGlass.new({
    Name = "KnoxHub",
    ConfigurationSaving = true
})

Tabs

Create navigation buttons in the sidebar.

local MainTab = Window:CreateTab("Main")

Texts (Sections)

Display headers or text labels inside your tabs.

MainTab:CreateSection("This is a Section Label")

Toggles

Switch features on or off.

MainTab:CreateToggle({
    Name = "Auto Farm",
    CurrentValue = false,
    Flag = "farm",
    Callback = function(v) print(v) end
})

Buttons

Execute a one-time script action.

MainTab:CreateButton("Instant Kill", function()
    print("Action Executed")
end)

Sliders

Adjust numerical values like Speed or Power.

MainTab:CreateSlider({
    Name = "WalkSpeed",
    Min = 16, Max = 300, CurrentValue = 16,
    Flag = "ws",
    Callback = function(v) print(v) end
})

Dropdowns

Choose from a list of options.

MainTab:CreateDropdown({
    Name = "Select Zone",
    Options = {"Spawn", "Desert", "PVP"},
    Callback = function(v) print(v) end
})

Notifications

Send a feedback message to the player.

Window:Notify({
    Title = "KnoxHub",
    Content = "Library Loaded!",
    Duration = 5
})
