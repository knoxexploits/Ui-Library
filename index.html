Quick Start

RedGlass is a dark-themed, glass-effect UI library designed for high performance on Roblox mobile and desktop.

local RedGlass = loadstring(game:HttpGet("https://raw.githubusercontent.com/knoxexploits/Ui-Library/refs/heads/main/RedGalss"))()

local Window = RedGlass.new({
    Name = "KnoxHub",
    ConfigurationSaving = true
})

Tabs & Sections

Organize your features into logical categories.

local MainTab = Window:CreateTab("Main")
MainTab:CreateSection("Combat Settings")

Toggles

The standard way to handle On/Off boolean states.

MainTab:CreateToggle({
    Name = "Auto Farm",
    CurrentValue = false,
    Flag = "farm_01",
    Callback = function(val) print(val) end
})

Sliders

Perfect for WalkSpeed, JumpPower, or FOV settings.

MainTab:CreateSlider({
    Name = "WalkSpeed",
    Min = 16, Max = 300, CurrentValue = 16,
    Flag = "ws_cfg",
    Callback = function(v) print(v) end
})

Dropdowns

Let users select a specific option from a table.

MainTab:CreateDropdown({
    Name = "Select Zone",
    Options = {"Spawn", "Desert", "Volcano"},
    Callback = function(opt) print(opt) end
})

Notifications

Send a toast message to the bottom right of the screen.

Window:Notify({
    Title = "Loaded",
    Content = "Script is ready!",
    Duration = 5
})
